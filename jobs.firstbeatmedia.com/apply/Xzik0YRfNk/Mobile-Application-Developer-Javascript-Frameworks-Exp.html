  • Multiple Countries
  • Contracted

We are looking for a developer who is familiar with modern JavaScript frameworks to help us out with our mobile project.  Big bonus points if you have experience with the React Native framework for mobile applications.


Your daily duties:
- Develop mobile app projects for IOS and Android devices using up-to-date technologies;
- Provide support on projects from the start to launch and beyond;
- Finding innovative solutions for problems;

Your skillset:
- 2+ years experience developing mobile applications for Apple and/or Android devices;
- Know-how in fixing errors using debugging tools;
- Familiar with GIT and/or SVN;
- Problem solver, detail oriented;
- Flexible and quick learner;

Bonus skills:
- 1+ years experience React Native, ReactJS or another modern JavaScript framework (Angular, VueJS, Flutther ... etc.);
- Experience with Ecma Script;
- Experience with Redux or other Flux implementation ;
- Writing tests for your code;

Additionally, you must be:
- fluent in English (chat, email, video);
- provide us relevant projects (based on required skills) and details about your role for each of them;

Conditions:

- Work remotely;
- Working hours: Monday-Friday 9am-5pm EST (there is some flexibility here);
- Full-time employment, this is NOT a part-time job;
- Opportunity for growth and promotion;
- Please state your desired salary when applying;
